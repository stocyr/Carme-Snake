Carme-Snake

1.0


Das Spiel wird auf dem LCD der CARME Kits angezeigt, die Bedienung erfolgt über die Pfeiltasten an einem PC.
Das Interface PC <-> CARME erfolgt per RS232 auf den UART0 Port des CARME Kits.
