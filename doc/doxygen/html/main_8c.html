volatile enum direction snake_direction = '?'
 

Globale Variable, welche vom Interrupthandler über UART aktualisiert wird.

Die Variable ist das Interface zur UART Schnittstelle. Sie enthält die Richtung, in die die Schlange steuern soll. Wird per PC Tastatur gesteuert und dann per UART zum CARME Kit gesendet.

Bevor überhaupt eine Taste gedrückt wurde hat die Variable den Wert '?', damit man erkennen kann, wenn dann erstmals ein r, o, l oder u hereinkommt.

Definiert in Zeile 58 der Datei main.c.

Wird benutzt von main() und step_forward().

volatile int timer_irq_flag = 0
 

Globale Variable, welche vom Interrupthandler über den Timer gesetzt wird.

Alle 1ms wird das Flag vom Interrupt des Timers gesetzt. Damit werden genaue Zeitverzögerungen möglich.

Definiert in Zeile 64 der Datei main.c.

Wird benutzt von delay().
